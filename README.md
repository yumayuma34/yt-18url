# yt-18url

yt-dlpを使用し、YouTubeのURLから、googlevideo.comの動画配信サーバーURLを取得するためのWindows向けGUIツールです。  

<img width="1176" height="514" alt="image" src="https://github.com/user-attachments/assets/b837dd0a-c731-4c03-a149-0ae2fbf3b90d" />

---

## 使い方

1. リリースにあるZIPをダウンロードし、解凍してください。  
2. `yt-18url.exe` を実行します。
3. 初回起動時のみ、yt-dlpのダウンロードが求められます。また、更新がある場合は更新が求められます。
4. YouTube の動画 URL を入力して「取得」を押します。
5. 取得したURLが表示され、自動でクリップボードにコピーされます。

---

## yt-dlp の更新について

- 起動時に yt-dlp のバージョンを自動チェックします。
- yt-dlpがダウンロードされていない場合、もしくは古い場合のみ更新ダイアログが表示されます。
- 手動で更新確認をしたい場合は、「yt-dlpの更新を確認」をクリックして確認が行えます。
